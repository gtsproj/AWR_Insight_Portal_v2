🌐 Network Configuration
Configure the actual server IP or hostname so Grafana dashboard links work from other machines on the intranet. Leave as localhost for single-machine access only.
Example for intranet: Portal URL = http://192.168.1.100:8000  |  Grafana URL = http://192.168.1.100:3000
After saving, all Grafana dashboard links on the home page will use the configured URL. Remote users on the intranet can then open dashboards directly.
