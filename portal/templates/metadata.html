🖥️ Local AI Setup — Ollama Installation (Windows)

Ollama runs the AI model locally — no data leaves the server. Required for Local AI mode.

Installation Steps:
  1. Download Ollama for Windows ↗
  2. Run the installer (OllamaSetup.exe)
  3. Open Command Prompt and run:
    ollama pull llama3.1:8b
  4. Verify: ollama list
  5. In Portal → Settings → AI: set mode to Local AI, test connection
